Life is meant to be Enjoyed.


Flip Six Three Hole games is a Vancouver Island, B.C. based independent game developer. We strive to create games with exceptionally fun and enjoyable gameplay. Our first game is a multi-platform puzzle game called **AIOfficeEscape**. As a developer, we aim to release the best product we can make, utilizing our passion to create games that we ourselves want to play.
